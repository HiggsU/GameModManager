# Game Mod Manager Icons

Please add your game icon files here:

- `icon.png` - PNG icon for Linux/macOS (256x256 recommended)
- `icon.ico` - ICO icon for Windows (256x256 recommended)

You can create icons using tools like:
- [Favicon.io](https://favicon.io/)
- [IconArchive](https://www.iconarchive.com/)
- Or design your own!
